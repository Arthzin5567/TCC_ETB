# TCC_ETB
## Projeto do TCC do curso de Informática.

- Devo deixar claro que **não** fiz esse projeto sozinho, fiz ele com uma equipe no meu curso técnico, mas estou sem o contato deles no momento para dar os créditos. Prometo que vou listá-los aqui no futuro.
- Este é um projeto que simula algo real. Fizemos uma análise de requisitos numa clínica odontológica real para obter os dados necessários para a produção deste projeto. Caso ele fosse lançado de verdade, teria uma base de usuários sólida.
- A V1.0 é a versão original do site, enquanto a V1.1 é uma versão possuindo correções feitas com o auxílio da SonarQube para um trabalho acadêmico da faculdade.
